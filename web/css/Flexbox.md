# Flexbox

![flexbox](C:\Users\JIHYUN\TIL\web\css\flexbox.JPG)

- 배치 방향 설정
  - flex- direction
    - row(default)
    - row-reverse
    - column
    - column-reverse
- 메인축
  - justify-content
- 교차축
  - align-items(한줄), align-self(개별요소), align-content(여러줄)
- wrap(강제로 한줄에 배치할 지 결정)
  - nowrap(한줄배치 함)
  - wrap: 넘치면 다음줄
  - wrap-reverse: 넘치면 윗줄
- order: 순서, 음수 가능
- flex-glow: 남는 공간을 분배, 상대적 비율x, 음수x
