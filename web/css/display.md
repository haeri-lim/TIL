# display

- block
  - div / ul, ol, li / p / hr / form
  - 줄바꿈
  - 화면 전체의 가로폭 차지
  - 블록 레벨 요소 안에 인라인 요소 들어감
- inline
  - span / a / img / input, label / b, em, i, strong 
  - 줄바꿈x
  - content의 가로폭 차지
  - widtn, height, margin-top, margin-bottom 지정x
  - 상하 여백은 line-height로 지정
- inline-block
  - block과 inline요소 특징 전부 갖음
- none
  - 해당 요소 화면에 표시x (공간도 사라짐)
  - hidden은 요소가 공간 차지
